# The Jug Handle Inn — Website

A premium, modern-rustic, fully responsive **static website** for The Jug Handle Inn
(Cinnaminson, NJ). Built with **HTML5 + CSS3 + Vanilla JavaScript only** — no frameworks,
no build step required to deploy.

---

## Quick start

This is a plain static site. To preview locally:

```bash
cd jughandle-inn
python3 -m http.server 8000
# then open http://localhost:8000
```

To deploy, upload the contents of `jughandle-inn/` (everything **except** the
`build-scripts/` folder) to any static host — Bluehost, Cloudflare Pages, Netlify, etc.

## Replacing the placeholder images

Every image is a branded **SVG placeholder** sized to its slot, so the layout already
looks finished. To use real photos, just drop a file with the **same name** into the
same folder (you can switch the extension to `.jpg`/`.webp` and update the `src` in the
HTML, or keep the `.svg` name).

- **Food / hero / section photos** → `assets/images/general/` and `assets/images/food/`
- **Gallery** → `assets/images/gallery/gallery-1…9`
- **Event flyers** → `assets/images/events/event-1…8`
  - Square (1024×1024 or 1254×1254). To add/swap an event, change the image and the
    matching card data in `build-scripts/build_pages.py` (`build_events()`), or edit
    `events.html` directly.
- **Favicons & social share image** → `assets/images/logo/`

## How the dynamic Open/Closed status works

`assets/js/main.js` holds an `HOURS` table and computes status against the visitor's
**local time** every 60 seconds (it correctly handles the Thu–Sat close after midnight).
The status pill appears in the header, footer and on the contact page. If the inn's hours
change, edit the `HOURS` object in `main.js` and the hours table in `contact.html`.

## Editing content

Two options:

1. **Edit the HTML directly** — it's clean, semantic and commented.
2. **Regenerate from the scripts** (recommended for bulk/menu changes). The site was
   generated by the Python files in `build-scripts/`:
   - `build_shell.py` — business constants, SEO/meta, header, footer, icons
   - `build_menu.py` — both menus typed out as structured data + the music schedule
   - `build_pages.py` — every page's content

   ```bash
   cd build-scripts
   cp -r ../assets .          # scripts write next to assets
   python3 build_pages.py     # regenerates all 11 .html files
   ```
   (Scripts have **no third-party dependencies** — standard library only.)
